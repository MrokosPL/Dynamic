#Тестовое задание "Библиотека"
Проект представляет собой приложение для работы библиотеки.
Функции:
-Создание/редактирование пользователе
-Создание/редактирование книг
-Выдача книг пользователям
-Выгрузка всех выданных книг в виде JSON

#Эндпоинты:
Эндпоинты с описанием можно посмотреть в swagger по ссылке после запуска
http://localhost:8080/swagger-ui/index.html#/

Запуск:
Для запуска - запустить в ide или собрать jar-ник предварительно запустив postgres.
Порт стандартный 5432.
user = postgres
password = root
(меняется через properties)
